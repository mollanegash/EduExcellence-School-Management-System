# EduExcellence School Management System

A high-performance School Management platform developed using **Java** and **Spring Boot**. The system leverages a **Microservice-ready architecture** with a focus on speed and data reliability.

### 🚀 Key Features & Tech Stack
* **Backend:** Spring Boot, Spring Web, Spring Data JPA.
* **Performance Layer:** Integrated **Redis** caching to optimize student retrieval and reduce database load.
* **Persistence:** **H2 (In-memory database)** for rapid development and testing.
* **Operations:** Full CRUD (Create, Read, Update, Delete) functionality for student management.
